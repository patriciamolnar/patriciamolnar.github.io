Charity Website

This bilingual website was created using HTML5, CSS3, JavaScript and PHP. It is cross browser compatible, including Internet Explorer.

The website features a contact form that I hand-coded with PHP. It sends emails to the inbox located on the hosting server, which then get forwarded to the client's Gmail account, so she can send and receive emails directly from her Gmail account.

It also makes use of PHP Sessions and JavaScript cookies.

Additionally, in Chrome, the header features an interactive animation that makes the stars grow and shrink based on whether the user is hovering over them.

As the website is live, the source code is not available to the public.

Police API

This project was built using jQuery and Sass, and makes use of two rest APIs. It is compatible with older browsers, such as Internet Explorer.

A user can enter their British postcode and select a month. Based on this input, this single page application will return data on Police stop and searches in their area. The information shown includes the reasons for the searches and their resolution (e.g. whether the suspect was arrested).
